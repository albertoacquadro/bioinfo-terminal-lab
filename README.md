# bioinfo-terminal-lab

## Avvio rapido (Gitpod Ona)
Apri il laboratorio:
https://app.gitpod.io/#https://github.com/albertoacquadro/bioinfo-terminal-lab

### Prima volta (creazione environment)
1) Accedi con **GitHub** (se richiesto).
2) Se ti compare la schermata Home di Ona con il pulsante **New Environment**:
   - clicca **New Environment**
   - scegli/incolla il repository: `albertoacquadro/bioinfo-terminal-lab`
   - conferma e attendi la creazione dell’ambiente (cloning + dev container).

### Apri il terminale
In alto a destra clicca **Open VS Code** (se non è già aperto), poi apri il terminale:
- menu **Terminal → New Terminal**
- oppure tab **TERMINAL** (in basso).
